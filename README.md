## Overview

This project presents a LoRa-based IoT monitoring system using ESP32 and RYLR998 modules for long-range wireless communication. The system monitors water levels and gas leakage in real time and provides alerts through LCD display, buzzer, and Blynk cloud integration.

---

## Features

* Long-range wireless communication using LoRa
* Real-time water level monitoring
* Gas leakage detection
* ESP32-based dual node architecture
* LCD and buzzer alert system
* Blynk cloud monitoring
* Low power consumption
* Wireless smart monitoring system

---

## Hardware Components

| Component         | Specification |
| ----------------- | ------------- |
| ESP32             | WROOM-32      |
| LoRa Module       | Reyax RYLR998 |
| Gas Sensor        | MQ-2          |
| Ultrasonic Sensor | HC-SR04       |
| Display           | 16x2 I2C LCD  |
| Communication     | LoRa 865 MHz  |
